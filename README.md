# Financial Risk Assessment Dashboard Using Power BI

This project presents a financial risk assessment dashboard built using Power BI to analyze customer demographics, income patterns, and credit risk indicators.

## Dashboard Preview

![Dashboard](financial_risk_dashboard_preview.png)

## Key Insights

- Gender-based financial analysis
- Credit amount vs income relationship
- Income type distribution
- Customer risk segmentation
- Credit trends by vehicle ownership age

## Tools Used

- Microsoft Power BI
- Data Visualization
- Financial Risk Analysis

## Features

- Interactive filtering by gender
- Credit vs income analysis
- Customer income type segmentation
- Financial trend visualizations

## How to Use

1. Download the `.pbix` file
2. Open in Microsoft Power BI Desktop
3. Explore interactive insights
